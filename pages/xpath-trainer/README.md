# XPath Trainer 🎯

Un outil d'apprentissage interactif pour maîtriser XPath à travers 9 séries d'exercices progressifs.

## 📋 Description

XPath Trainer est une plateforme pédagogique complète proposant **54 exercices pratiques** répartis en 9 séries de difficulté croissante. Chaque série présente un DOM HTML réaliste (boutique e-commerce, dashboard analytique, formulaires, système de tickets, documentation technique...) avec 6 exercices à résoudre.

Le projet permet de développer des compétences sur :
- 🧭 **Axes de navigation** : `ancestor::`, `parent::`, `following-sibling::`, `preceding::`, `following::`
- 🎯 **Prédicats imbriqués** avec conditions multiples
- 📝 **Fonctions de texte** : `normalize-space()`, `contains()`, `starts-with()`
- 🔍 **Expressions XPath complexes**

## ✨ Fonctionnalités

- ✅ Validation automatique des réponses XPath
- 💡 Système d'indices progressifs pour chaque exercice
- 📚 Solutions complètes disponibles
- 🧪 Test des expressions XPath directement dans le navigateur
- 📊 Progression du niveau débutant au niveau avancé
- 🎨 Interface moderne et responsive

## 🚀 Installation et Utilisation

### Prérequis

Un navigateur web moderne (Chrome, Firefox, Safari, Edge)

### Lancement du projet

1. Clonez ce dépôt :
```bash
git clone https://github.com/VOTRE_USERNAME/XPath_Trainer.git
cd XPath_Trainer
```

2. Ouvrez `index.html` dans votre navigateur :
   - Double-cliquez sur le fichier `index.html`, ou
   - Utilisez un serveur local (ex: Live Server pour VS Code)

3. Commencez par l'exercice 1 et progressez à votre rythme !

## 🎓 Parcours d'Apprentissage

Les exercices sont organisés en 9 niveaux de difficulté progressive :

1. **Niveau 1** - Sélection Simple : Bases de XPath
2. **Niveau 2** - Navigation dans la hiérarchie : Axes parent, enfant, sibling
3. **Niveau 3** - Prédicats et filtres simples
4. **Niveau 4** - Fonctions de texte et chaînes
5. **Niveau 5** - Axes complexes et navigation avancée
6. **Niveau 6** - Prédicats imbriqués et conditions multiples
7. **Niveau 7** - Combinaison d'axes et de fonctions
8. **Niveau 8** - Expressions XPath complexes
9. **Niveau 9** - Défis avancés et cas d'usage réels
