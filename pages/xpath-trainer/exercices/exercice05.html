Objectifs Pédagogiques

Dans cet exercice, vous allez apprendre à :

  • Utiliser parent:: pour remonter au parent direct
  • Utiliser ancestor:: pour remonter à n'importe quel ancêtre
  • Combiner les axes ascendants avec des prédicats
  • Naviguer dans une structure de tableau HTML complexe
Niveau Moyen Série 5

🎯 DOM Cible - Système de Suivi de Tickets (Redmine)

Voici un tableau de tickets issu d'un système de gestion de projets. Utilisez les axes ascendants pour localiser les éléments demandés.

# Projet Tracker Statut Priorité Sujet Assigné à Actions
81 Gestion de parking Bug New Normal Anomalie de calcul de tarif User 56
80 Gestion de parking Bug New High Erreur de validation formulaire User 55
79 Plateforme Web Feature In Progress Normal Ajouter système de paiement User 46
78 E-Commerce Bug New Urgent Problème de connexion base de données User 52
77 Mobile App Bug Resolved Normal Crash au démarrage sur iOS User 48
76 Gestion de parking Feature In Progress Normal Interface de réservation en ligne User 45

Légende des priorités :

  • Normal - Priorité standard
  • High - Priorité élevée
  • Urgent - Nécessite une action immédiate

📝 XPath à Trouver

Trouvez le XPath qui cible UN seul élément précis en utilisant les axes ascendants.

Exercice 1 : Bouton de la ligne avec l'ID 78

Trouvez le bouton "Modifier" de la ligne dont le premier lien (colonne ID) a le href "/issues/78".
💡 Partez du lien avec @href='/issues/78', remontez à la ligne, puis descendez vers le bouton.

Exercice 2 : Trouver la ligne avec priorité "High"

Trouvez la ligne (tr) ancêtre d'une cellule contenant exactement "High" dans la colonne priorité.

Exercice 3 : Remonter au tableau depuis une cellule

À partir du lien dont le href est "/issues/78", remontez jusqu'au tableau (table) ancêtre.

Exercice 4 : Ligne avec un Bug de priorité Urgent

Trouvez la ligne (tr) qui contient un ticket de type "Bug" et de priorité "Urgent".
💡 Utilisez des prédicats pour filtrer sur tracker ET priorité, puis remontez.

Exercice 5 : Cellule parent d'un lien de projet

Trouvez la cellule (td) parente du lien vers le projet "E-Commerce" (href="/projects/ecommerce").

Exercice Bonus : Ligne avec statut "Resolved"

Trouvez la ligne (tr) ancêtre d'une cellule de statut contenant "Resolved" et qui a l'ID "issue-77".
💡 Combinez ancestor:: avec un prédicat sur l'attribut @id du tr.
