Objectifs Pédagogiques

Dans cet exercice, vous allez apprendre à :

  • Utiliser text() pour sélectionner par le contenu textuel exact

  • Différencier les espaces et la casse dans les sélecteurs
  • Localiser des liens de navigation spécifiques

📝 XPath à Trouver

Trouvez le XPath qui cible UN seul élément précis en utilisant le texte exact.

Exercice 1 : Sélectionner le lien "Services"

Trouvez le lien du menu principal qui contient exactement le texte "Services".

Exercice 2 : Sélectionner le bouton "Créer un compte"

Trouvez le bouton qui contient exactement le texte "Créer un compte".

Exercice 3 : Sélectionner le titre "Nos Services"

Trouvez le titre h3 qui contient exactement "Nos Services".

Exercice 4 : Sélectionner "Formation" dans le breadcrumb (fil d'Ariane)

Trouvez le span dans le fil d'Ariane qui contient exactement "Formation".
💡 Attention : il y a aussi "Formation" dans le sous-menu !

Exercice 5 : Sélectionner le titre de section "Support"

Trouvez le h4 dans le footer qui contient exactement "Support".

Exercice Bonus : Sélectionner le lien "Développement sur mesure"

Trouvez le lien du sous-menu qui contient le texte "Développement sur mesure".
💡 Le texte est dans le lien, mais il y a aussi une icône avant !
