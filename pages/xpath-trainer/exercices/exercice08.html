Objectifs Pédagogiques

Dans cet exercice, vous allez apprendre à :

  • Utiliser following:: pour tous les éléments suivants
  • Utiliser preceding:: pour tous les éléments précédents
  • Combiner plusieurs axes dans un même XPath
  • Naviguer dans une structure de documentation complexe
Niveau Difficile Série 8

🎯 DOM Cible - Documentation Technique

Voici une structure de documentation technique avec sections, code, notes et avertissements. Utilisez les axes complexes pour naviguer.

📚 Guide d'Installation

🔧 Prérequis

Avant de commencer l'installation, assurez-vous d'avoir les éléments suivants :

  • Node.js version 18 ou supérieure
  • NPM version 9 ou supérieure
  • Git installé sur votre système
💡 Utilisez nvm pour gérer vos versions de Node.js

📦 Installation

Suivez ces étapes pour installer le projet :

Étape 1 : Cloner le dépôt

git clone https://github.com/example/project.git
cd project
⚠️ Assurez-vous d'avoir les droits d'accès au dépôt

Étape 2 : Installer les dépendances

npm install
💡 L'installation peut prendre quelques minutes

Étape 3 : Configuration

cp .env.example .env
npm run setup
🚨 Ne committez jamais votre fichier .env dans Git

⚙️ Configuration

Configurez les paramètres de votre application :

Base de données

DATABASE_URL=postgresql://localhost:5432/mydb
DATABASE_POOL_SIZE=10
⚠️ Modifiez ces valeurs selon votre environnement

Serveur Web

PORT=3000
HOST=localhost
💡 Le port par défaut est 3000 mais peut être modifié

🧪 Tests

Exécutez les tests pour valider l'installation :

Tests unitaires

npm test

Tests d'intégration

npm run test:integration
🚨 Les tests d'intégration nécessitent une base de données active

🚀 Déploiement

Prêt pour la production ? Suivez ces recommandations :

Build de production

npm run build
npm run start:prod
✅ Le build optimise automatiquement votre code

📝 XPath à Trouver

Trouvez le XPath qui cible UN seul élément précis en utilisant les axes complexes.

Exercice 1 : Note warning après un h3 spécifique

Trouvez la div.note.warning qui suit le h3 contenant "📦 Installation" et qui contient le texte "Assurez-vous d'avoir les droits d'accès au dépôt".

Exercice 2 : Note danger après un code spécifique

Trouvez la div.note.danger qui suit le bloc code contenant "npm run test:integration".

Exercice 3 : Élément précédant une note spécifique

Trouvez le h4 qui précède la div.note contenant "Ne committez jamais votre fichier .env".

Exercice 4 : Note info dans une section spécifique

Trouvez la div.note.info qui se trouve dans la section ayant l'id "installation" et qui contient "L'installation peut prendre quelques minutes".
💡 Partez de la section et descendez avec .// pour chercher à l'intérieur.

Exercice 5 : Code suivant un avertissement

Trouvez le premier élément pre.code qui suit une div.note.warning contenant "Modifiez ces valeurs selon votre environnement".

Exercice Bonus : Navigation complexe multi-axes

Trouvez la div.note.success qui se trouve après toutes les div.note.danger du document.
💡 Utilisez following:: depuis la dernière note danger (celle avec "base de données active").
