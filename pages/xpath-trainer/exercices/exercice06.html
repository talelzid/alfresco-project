Objectifs Pédagogiques

Dans cet exercice, vous allez apprendre à :

  • Utiliser following-sibling:: pour sélectionner les frères suivants
  • Utiliser preceding-sibling:: pour sélectionner les frères précédents
  • Combiner les axes siblings avec des prédicats
  • Naviguer dans une structure de formulaire complexe

📝 XPath à Trouver

Trouvez le XPath qui cible UN seul élément précis en utilisant les axes siblings.

Exercice 1 : Frère suivant d'un label

Trouvez l'input qui est le frère suivant immédiat du label "Prénom".

Exercice 2 : Section "Adresse" après "Informations personnelles"

Trouvez la div.form-section qui suit "personal-info" et qui contient un titre h3 "Adresse".

Exercice 3 : Message d'aide après le champ "Rue"

Trouvez le span.help-text qui est frère suivant de l'input avec id="street".

Exercice 4 : Section "Adresse" précédant "Préférences"

Trouvez la div.form-section qui précède "preferences" et qui contient un titre h3 "Adresse".
💡 Utilisez preceding-sibling avec un prédicat sur le contenu.

Exercice 5 : Bouton suivant le bouton "Réinitialiser"

Trouvez le bouton qui suit immédiatement le bouton de type "reset".

Exercice Bonus : Checkbox précédé d'un autre checkbox

Trouvez l'input checkbox avec id="notifications" qui a un frère précédent checkbox.
💡 Combinez @id avec preceding-sibling::input[@type='checkbox'].
